Create RemindersListScreen (list all reminders across vehicles, filter active/overdue, delete) and ProfileScreen (view/edit first name, last name).

## Purpose
These are the two most immediately missing screens for any owner — they can add reminders and set up a profile but cannot view or manage either.

## Output
Two new screen pairs (Screen + ScreenModel), repository extensions for getAllReminders and updateProfile, DI registration.
